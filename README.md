## 🚀 Welcome to Pizzathon!

**Pizzathon** is the official open-source mobile application for the ultimate pizza hackathon! Built with **Flutter**

---

## ✨ Key Features
- 🏆 **Prize Tracking:** See the scoreboard and track your progress.
- 📸 **Gallery:** Upload your pizzas to win prizes.
- 🛠️ **Open Source:** Designed for the community, by the community. Feel free to contribute!

## 💻 Tech Stack

- **Frontend:** [Flutter](https://flutter.dev)
- **State Management:** Cubit & Bloc
